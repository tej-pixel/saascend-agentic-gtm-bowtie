How the agents actually run

🪡
Orchestration layer

Agents run on a workflow engine (n8n), not bolted inside your CRM. Triggers, retries, scheduling and logging live here — pause or swap any agent without touching your system of record.

🗄️
Your system of record

Salesforce / HubSpot stays the source of truth. Agents read context and write structured results back over the API on least-privilege scopes — no shadow database.

🧠
Reasoning + human-in-the-loop

Claude does the reasoning; you set the autonomy. Each agent is read-only, drafts-for-approval, or writes-back — configurable per field, with a human gate wherever you want one.

🔒
Security & observability

Scoped API tokens you own and can revoke, a full audit log of every action, and data that stays in your tenant. Nothing goes live without your sign-off.

🚦 Rollout: start with one agent → prove impact in a sandbox → expand. No big-bang. Most agents are live in 1–3 weeks.
